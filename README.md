# Black Training Center

Landing page moderna y adaptable desarrollada para un gimnasio con el objetivo de presentar sus servicios e información de contacto.

## Vista Previa

**Demo en línea:**  
https://black-training-center-web.vercel.app/

## Características

- Diseño responsive para computadoras, tablets y dispositivos móviles.
- Interfaz moderna orientada al ámbito fitness.
- Navegación fluida entre secciones.
- Presentación de servicios e información del gimnasio.
- Acceso a medios de contacto y redes sociales.
- Experiencia de usuario optimizada.

## Tecnologías

- React
- JavaScript
- HTML5
- CSS3
- EmailJS
- Vercel

## Objetivos

El objetivo de este proyecto fue practicar desarrollo frontend mediante la creación de una landing page para un caso de uso real, enfocándome en:

- Arquitectura basada en componentes.
- Diseños responsivos.
- Principios de diseño de interfaces.
- Navegación intuitiva para el usuario.

## Lo que Aprendí

A través de este proyecto mejoré mis conocimientos en:

- Desarrollo de componentes con React.
- Diseño web responsive.
- Organización y estructura de proyectos frontend.
- Despliegue y alojamiento de aplicaciones con Vercel.
- Creación de interfaces para escenarios de negocio reales.
- Uso e integración de librerías externas.

## Mejoras Futuras

- Registro de membresías en línea.
- Gestión de horarios y clases.
- Integración con un backend.
- Contacto mediante WhatsApp.
- Panel de administración.
